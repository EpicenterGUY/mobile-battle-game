# 테스트 체크리스트

현재 분리된 파일 구조를 유지한 상태에서 아래 항목을 기준으로 테스트를 진행합니다.

## 1) 싱글 AI 테스트

- [ ] AI가 전투 시작 시 정상적으로 스폰/초기화된다.
- [ ] AI가 플레이어를 정상적으로 탐지하고 추적한다.
- [ ] AI의 기본 공격(또는 스킬) 쿨타임이 의도대로 동작한다.
- [ ] AI가 피격 시 올바른 반응(경직, 데미지 반영, 상태 전이)을 보인다.
- [ ] AI 체력이 0이 되면 정상적으로 사망 처리 및 정리(오브젝트/이펙트)가 된다.
- [ ] 싱글 플레이 중 프레임 저하/메모리 급증이 없는지 확인한다.

## 2) 온라인 멀티 테스트

- [ ] 2인 이상 접속 시 룸 생성/입장이 정상 동작한다.
- [ ] 플레이어 위치/애니메이션/행동 동기화가 일정한지 확인한다.
- [ ] 공격 판정 및 피격 결과가 클라이언트 간 일관되게 반영된다.
- [ ] 지연(latency) 환경에서 보정(보간/예측)이 과도한 튐 없이 동작한다.
- [ ] 재접속 시 상태 복구(위치, HP, 진행 상태)가 정상 동작한다.
- [ ] 한 명 이탈 시 나머지 세션이 안정적으로 유지된다.

## 3) 서브스킬 테스트

- [ ] 각 캐릭터/클래스의 서브스킬이 정상적으로 발동된다.
- [ ] 서브스킬 자원(마나/에너지/스택) 소모가 정확하다.
- [ ] 서브스킬 쿨타임 UI 및 실제 내부 쿨타임이 일치한다.
- [ ] 버프/디버프형 서브스킬의 지속시간 및 중첩 규칙이 의도대로 동작한다.
- [ ] 서브스킬 발동 중 이동/피격/취소 가능 여부가 기획과 일치한다.
- [ ] 네트워크 환경에서 서브스킬 이펙트 및 판정 동기화가 유지된다.

## 4) 로테이션 테스트 체크리스트

- [ ] 기본 공격 → 서브스킬 → 회피 → 궁극기 순환이 끊김 없이 실행된다.
- [ ] 로테이션 반복 시 쿨타임 공백 구간이 기획 범위 내인지 확인한다.
- [ ] 로테이션 중 자원 수급/소모 밸런스가 의도대로 유지된다.
- [ ] 우선순위 스킬(생존/딜링)이 상황에 맞게 자연스럽게 선택된다.
- [ ] 장시간(예: 10분 이상) 반복 시 입력 누락/상태 꼬임이 발생하지 않는다.
- [ ] 로테이션 수행 결과(DPS/클리어 타임 등)가 기준값 내에 들어오는지 확인한다.
